# ⚡ JS Notes — Net Ninja Diary

> A cute, dark-themed coding notebook built for JavaScript learners following the [Net Ninja](https://www.youtube.com/@NetNinja) tutorial series.

---

## ✨ What is this?

**JS Notes** is a personal, browser-based note-taking app made specifically for jotting down JavaScript concepts as you learn them. No sign-up, no installs — just open and start writing!

Built with pure HTML, CSS & JavaScript. All notes are saved directly in your browser via `localStorage`.

## 🎯 Features

- 📝 **Rich code editor** with line numbers and tab support
- ⚡ **Quick-insert snippets** — `const`, `let`, `function`, `console.log`, `forEach`, `map`, `filter`, `querySelector`, `addEventListener` and more
- 🏷️ **Topic tags** — organise notes by Variables, Functions, Arrays, Loops, DOM, Events, Objects
- 💾 **Auto-saves** 2 seconds after you stop typing
- 🔍 **Search** through all your notes instantly
- 📊 **Word, line & char count** in the status bar
- ⌨️ **Keyboard shortcut** — `Ctrl+S` / `Cmd+S` to save manually
- 🌌 **Dark starfield theme** — easy on the eyes during long study sessions
- 🗑️ **Delete notes** when you're done with them
- 💯 **Zero dependencies** — no frameworks, no backend, no accounts

---

## 🛠️ How to Use

1. Click **+ New Note** to create your first note
2. Give it a title (e.g. `"Arrow Functions"`)
3. Pick a **topic tag** from the dropdown
4. Write your notes in the editor — use the quick-insert buttons for common JS patterns
5. Notes **auto-save** — or hit `Ctrl+S` to save manually
6. Use the **search bar** to find any note later
